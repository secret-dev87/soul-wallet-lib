# soul-wallet-lib
The EIP-4337 interaction library for SoulWallet

## Usage


```bash
npm install --save-dev git+https://github.com/proofofsoulprotocol/soulwalletlib.git#v0.0.1
```

```javascript
import { WalletLib } from 'soul-wallet-lib';
```
